# 프로젝트 개요 (Project Brief)

## 1. 프로젝트 목표  
- **핵심 문제**: 프로토타입 단계에서 서버 비용 없이 데이터를 관리하고 팀원들과 공유할 수 있는 시스템이 필요함
- **최종 목표**: GitHub Repository를 데이터베이스처럼 활용하는 Streamlit 기반의 AI 스마트 행정 플랫폼 구축
- **기대 효과**: 
  - 서버 비용 절감 (GitHub 무료 플랜 활용)
  - 데이터 버전 관리 및 협업 용이
  - 빠른 프로토타입 개발 및 배포

## 2. 핵심 기능  
- **통합 대시보드**: 재학생 현황, MOU 현황 등 주요 지표 시각화 (Plotly 차트)
- **주간보고 AI 챗봇**: 키워드 기반 주간보고서 검색 및 채팅 인터페이스
- **스마트 일정 관리**: 직원별 일정 조회 및 공통 빈 시간 찾기
- **직원 추천 시스템**: 전문성, 관심사 기반 직원 검색
- **기관평가 코칭**: 평가 항목별 가이드라인 및 전년도 피드백 제공
- **명함 공유 허브**: 외부 기관 담당자 정보 검색 및 관리
- **데이터 관리 (Admin)**: JSON 파일 업로드 및 GitHub 저장 기능

## 3. 타겟 사용자  
- **주요 사용자**: KDI 대학원 행정 직원
- **사용 목적**: 
  - 일상 업무 데이터 관리 및 공유
  - 보고서 검색 및 정보 조회
  - 일정 조정 및 협업
  - 기관평가 준비
